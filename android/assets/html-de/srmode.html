In diesem Modus arbeitet der ESR Scanner wie ein Belegleser (z.B. paypen). Einfach den Einzahlungsschein einscannen und schon wird es (über WLAN) an deinen Computer gesendet.

Anleitung
  1. Downloade den ESR Receiver (es wird Java benötigt) von http://www.wuala.com/gulb/open/ESRReceiver
  2. Starte das WLAN, WLAN Tether, Bluetooth Tether oder USB Tether auf deinem Smartphone
  3. Stelle sicher, dass der Computer, auf dem der ESR Receiver läuft, eine direkte Netzwerkverbindung zum Smartphone hat. Dies kann über eine der oben erwähnten Möglichkeiten sein.
  4. Starte den ESR Receiver
  5. Drücke um den Beleglesermodus zu aktivieren
  6. Trage die IP Adresse, die jetzt im ESR Scanner unten rechts angezeigt wird, im ESR Receiver ein und drücke "Connect"
  7. Oder (ab ESR Receiver version 0.6.1) Warte (max. 1 Minute), bis der ESR Receiver sich selber verbindet
  8. Warte bis der ESR Receiver den Status auf "Connected" setzt
  9. Plaziere den Cursor in das Feld, wo die Kodierzeile eingetragen werden soll.
  10. Scannne die Zahlung

Bei aktiviertem Beleglesermodus wird die gescannte Codierzeile nicht gespeichert.

Um Codierzeilen von bereits gescannten Zahlungen zu senden, muss man den Beleglesermodus deaktivieren, den Verlauf öffnen und dort drücken.
