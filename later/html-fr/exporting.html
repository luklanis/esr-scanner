Pour exporter les bulletins de dépôt en des ordres de paiement, le Scanner ESR a besoin de ton IBAN et d'au moins deux lignes de ton adresse. Il suffit de remplir les champs ci-dessous dans les réglages.

Sur et ensuite « DTA » le fichier est alors créé et transmis à l'application sélectionnée.

Dans l'historique, les paiements individuels peuvent être sélectionnés. S'ils ont déjà été exportés, le nom du fichier qui contient le bulletin de dépôt apparaît ci-dessous. Chaque bulletin de dépôt n'est exporté qu'une seule fois. Cependant, s'il doit être exporté une deuxième fois, appuyez sur le nom de fichier.
