Le scanner ESR propose deux solutions pour soutenir ceux qui veulent traiter leurs paiements directement avec un périphérique Android.

  • Copie des notifications
    Lors de la fermeture du scanner ESR, une entrée apparaît dans les notifications. Cette entrée vous permet de passer facilement entre une activité (par exemple, le navigateur avec le portail de services bancaires en ligne ouvert) et le scanner ESR. Si vous ouvrez l'activité qui a besoin de données du scanner ESR, cliquez sur l'entrée dans les notifications et le scanner ESR démarre. Comparé à son comportement habituel, le scanner ESR se ferme automatiquement après un balayage réussi et retourne au navigateur. Il a ainsi scanné la ligne de code ou le numéro de référence (réglable) et l'a copié dans le presse-papiers. Maintenant, vous devez seulement insérer le texte copié dans le champ souhaité dans le navigateur.

    Pour pouvoir utiliser ceci, vous devez cocher « Copie le résultat de la notification » dans les paramètres du scanner ESR.

  • Scanner ESR comme méthode de saisie
    Le scanner ESR s'inscrit parmi les méthodes de saisies installées. Pour utiliser le scanner ESR en tant que méthode de saisie, l'on doit d'abord l'autoriser et l'activer dans les paramètres du système. Au lieu d'un clavier virtuel, le scanner ESR apparaît et insère, après un balayage réussi, cette même ligne de code dans la boîte de texte sélectionnée. Malheureusement, le scanner ESR ne fonctionne seulement en orientation paysage.

    Pour pouvoir retourner facilement au clavier tactile, il y a un bouton en bas lorsque vous tenez l'appareil en orientation verticale (en mode portrait).
