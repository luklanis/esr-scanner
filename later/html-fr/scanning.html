En haut à gauche de l'écran est indiqué quelle partie du code sera nécessaire pour la prochaine étape.

Le Scanner ESR scanne en permanence tout ce qui apparaît dans le carré blanc. Pour analyser la ligne de code, il faut maintenir au moins la partie de la ligne qui est actuellement nécessaire à l'intérieur du cadre blanc :

   

Lorsque vous essayez de scanner un bulletin de versement rouge, les parties nécessaires sont automatiquement reconnues, dès qu'au moins la première partie a été saisie avec succès.

Parfois, l'ensemble de la linge de code est détectée en une seule étape. Si une seule partie a été saisie, la prochaine partie requise est affichée. Après chaque enregistrement, un bip retentit. Le résultat final comprend au moins le numéro de compte et le numéro de référence. Si le montant n'est pas spécifié, vous devez le saisir vous-même. Pour avoir le contrôle sur les paiements effectués, il est également recommandé de saisir le nom du destinataire. Le nom du destinataire du paiement est sauvegardé pour des paiements ultérieurs avec le même numéro de compte.

Si vous avez des problèmes avec le scan, immobilisez le téléphone autant que possible. Si le téléphone a des difficultés à faire la mise au point, tenez le téléphone plus proche ou plus loin ou essayez en mode macro.

IMPORTANT : Il faut toujours comparer les résultats avec le bulletin de dépôt. Le Scanner ESR vérifie chacun des trois montants de contrôles, cependant des erreurs peuvent se produire ! En cas d'erreur lors du scan, le paiement peut être effacé en le maintenant appuyé longtemps dans l'historique.
