In this mode ESR Scanner works like a payment slip reader (ex. paypen). Simply scan your payment and the code row will be sent to your computer.

How to use
  1. Download ESR Receiver (you need Java) from http://www.wuala.com/gulb/open/ESRReceiver
  2. Your
  3. Ensure that the computer, that runs the ESR Receiver, is directly connected to the smartphone over a network solution listed above.
  4. Start the ESR Receiver
  5. Press to activate payment slip reader mode
  6. Fill in the showed (on the bottom right corner) ip address on ESR Receiver and press "Connect"
  7. Or (since ESR Receiver version 0.6.1) Wait (max. 1 minute), until the ESR Receiver connects automatically
  8. Wait until ESR Receiver becomes connected state
  9. If "Auto paste" on the ESR Receiver is checked, just set the cursor to the location where the code row should be placed and scan the code.
  10. Otherwise scan the code and paste it with Ctrl-V or right-click-paste in the input field.

If you have slip-reader mode activated, the code row wouldn't be saved on the phone.

If you'd like to send code rows from already scanned payments, deactivate slip-reader mode, open History and press .
