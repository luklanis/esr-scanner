Dans ce mode, le scanner ESR fonctionne comme un lecteur de documents (par exemple, PayPen). Il suffit de scanner le bulletin de dépôt, puis il est envoyé (via WiFi) sur votre ordinateur.

Tutoriel
  1. Télécharge le récepteur ESR (il nécessite Java) depuis http://www.wuala.com/gulb/open/ESRReceiver
  2. Démarrez le WIFI, WIFI affilié, Bluetooth affilié ou USB affilié sur ton smartphone
  3. Assure-toi que l'ordinateur sur lequel le récepteur ESR est installé dispose d'une connexion réseau directe avec le smartphone. Ceci peut-être l'une des possibilités évoquées ci-dessus.
  4. Démarre le récepteur ESR
  5. Presse pour activer le mode de lecteur de document,
  6. Saisis l'adresse IP qui est désormais affichée dans le coin inférieur droit du scanner ESR dans le récepteur ESR et appuie sur « Connecter »
  7. Ou (avec un récepteur ESR à partir de la version 0.6.1) Attend (max 1 minute) jusqu'à ce que le récepteur ESR se connecte automatiquement
  8. Attends que le récepteur ESR se mette en statut « Connecté »
  9. Lorsque « Coller automatiquement » est sélectionné dans le récepteur ESR, positionne le curseur dans le champ où la ligne de code doit être saisie et scanne le paiement.
  10. Si non, alors il suffit de scanner le paiement, puis d'ajouter la ligne de code à l'emplacement désiré en utilisant Ctrl-V ou clic-droit et coller.

Lorsque le mode lecteur de documents est activé, la ligne de code scannée n'est pas enregistrée.
